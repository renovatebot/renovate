Extracts dependencies from [vcpkg](https://learn.microsoft.com/vcpkg/) `vcpkg.json` manifest files. Each entry under `dependencies[]` and `overrides[]` is emitted as a `PackageDependency` wired to the [`vcpkg`](../../datasource/vcpkg/readme.md) datasource and [`vcpkg`](../../versioning/vcpkg/readme.md) versioning.

`dependencies[]` entries carry a lower-bound constraint via the `version>=` field; bare-string entries and objects without `version>=` are emitted with `skipReason: 'unspecified-version'` because vcpkg has no concrete version to update. `overrides[]` entries pin an exact version via one of `version`, `version-semver`, `version-date`, or `version-string`; a non-zero `port-version` is appended as `#N` to match the upstream port-version encoding documented at [vcpkg.json reference](https://learn.microsoft.com/vcpkg/reference/vcpkg-json).
